🎛️ Редактор seance.json

Ціни + кольори + ложі + сегменти рядів (один сеанс)

🧾 Дані сеансу

Назва вистави — для глядача. Seance ID — технічний ключ. Poster URL — пряме посилання на афішу PNG зі Storage.

➕ Правило по рядах

Для суцільних рядів. Наприклад: P1-3, P4-8.

🎨 Діапазон місць у рядах

Для випадків типу: ряд 7, місця 1–6 одна ціна/колір, 7–17 інша, 18–23 третя. Записується в seat_overrides автоматично.

🎭 Ложі

Ложа А = A0-M1...A0-M18, Ложа Б = B0-M1...B0-M18. Також записується в seat_overrides.
